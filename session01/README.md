# Session 01

Small code that makes a led light blink.

## Specifications

Raspberry: `Raspberry Pi 3` <br />
Language: `C` <br />

## Usage

In the raspberry, open the command prompt and type:

### To Compile

```
& gcc blink.c -o outputname -lwiringPi
```

### To Run

```
& sudo ./outputname // in my case, also blink
```
