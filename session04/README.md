# DHT11 Temperature & Humidity Sensor

Simple code in c that outputs to the terminal the humidity and temperature from a DHT11 sensor connected to the Raspberry Pi 3. <br />

## Specifications

Language: `C`<br />
Sensor: `DHT11` or `ARD-360` <br />
